# BOE Analysis — Real Estate Investment Tool

A Python-based real estate "Back-of-the-Envelope" (BOE) analysis tool for quickly assessing the viability of potential investment properties.

## What It Does

- Models cash flow, NOI, cap rate, and ROI for a property
- Runs quick deal pipeline analysis
- Helps filter investment opportunities before deep due diligence

## Stack

- Python
- Jupyter Notebook
- pandas

## Files

- `RE BOE Functions.ipynb` — core calculation functions
- `RE Draft Pipeline.ipynb` — deal pipeline analysis
